Les forums

Cette page permet de créer, modifier, déplacer, supprimer des forums ou des sous-forums.

Création d'un nouveau forum

Cette action va requérir de compléter les informations suivantes :

  • niveau d'accès : cocher les groupes d'autorisations qui seront autorisés à accéder à cette catégorie
  • position de la catégorie : en cas de pluralité de forums la valeur saisie sert à déterminer l'ordre de tri des différentes forums. Plus la valeur est proche de 0 (zéro) pour le forum sera positionnée vers le haut dans sa catégorie
  • indiquer ensuite le nom du forum et sa description
  • catégories : sélectionner dans la liste déroulante à quelle catégorie affecter ce forum
  • etat : actif - inactif ; permet de désactiver un forum
  • autoriser le html : une réponse positive permettra aux auteurs de messages d'utiliser du code html à l'intérieur du message, ce qui peut être dangereux en matière de sécurité
  • autoriser les signatures : en répondant oui la signature enregistrée dans le profil de l'auteur sera affichée
  • autoriser les sondages : si le module xoopspoll est installé et activé, les membres autorisés pour créer ou répondre à un sondage en fonction des autorisations qui leur sont accordées.
  • allouer un préfixe de sujet : préciser ici si vous souhaitez attribuer la capacité à utiliser un préfixe pour les sujets tel que paramétré dans les préférences du module
  • seuil des sujets chauds : ?????
  • autoriser les pièces jointes : permet aux auteurs de messages d'ajouter un fichier à la conversation
  • taille maximum en kb : taille maximum en kilobytes du fichier qui peut être uploadé
  • autoriser les extensions : définisser les extensions de fichiers autorisées à être uploadées
  • permissions : pour chaque situation proposée, cocher les groupes autorisés

Après validation de la création du forum, vous aurez accès à l'écran de saisie des modérateurs

Autres opérations

  • permet de modifier une catégorie ou un forum
  • supprime une catégorie ou un forum (et ce qu'ils contiennent)
  • déplace un forum ou sous-forum dans une autre catégorie
